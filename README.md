# 🍎 Fruit Runner

A 2D platformer game built with **Godot Engine**, inspired by classic side-scrollers like Mario. Guide your character through levels, collect fruits to score points, and dodge dangerous mushroom enemies.

## 🎮 Gameplay
- Run and jump across platformer levels
- Collect fruits to increase your score
- Avoid or defeat mushroom enemies
- Classic arcade-style challenge

## 🛠️ Built With
- [Godot Engine](https://godotengine.org/)-using c++ coding
- GDScript
- Exported as APK for Android

## 📱 Installation
1. Download the `.apk` file from the [Releases](../../releases) tab
2. Enable "Install from unknown sources" on your Android device
3. Install and play!
